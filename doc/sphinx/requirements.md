# Requirements

Since this package is based on the [ICOtronic package], the requirements for using the ICOtest package are [pretty much the same as the requirements for the ICOtronic package](https://mytoolit.github.io/ICOtronic/#requirements).

In addition, **some tests** also assume:

- that you connected a programming board to the hardware, and
- installed either {ref}`Simplicity Commander or Simplicity Studio <simplicity>`. For more information, please take a look at the section {ref}`Configuration <configuration>`.

[icotronic package]: https://icotronic.readthedocs.io
